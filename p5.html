P5: 探究学習

チェンジメーカー育成カリキュラム

P4で実践が始まった → P5は生徒の探究を伴走する

P4の事例研究会で、教員が教科学習や探究学習の多様な実践を共有し始めた。AIチューターを使った授業も、AI統合型の対話授業も、教員同士の集合知として育っている。

そのとき、次の問いが生まれる:

「生徒の探究を、どう伴走すればいいか？」

P5は、その問いに応える。 P3で定義した「両利きの学校」のうち、探究学習（探索）の側を具体的なカリキュラムとして設計する。このカリキュラムは、P3で示したように、企業のAI共創型新規事業開発プロセスを高校生向けに翻訳したものだ。

❶ 出会う——テーマとの邂逅

コンピテンシー

7. 意味の自己生成 / 5. 仮説駆動

窓講師カタログ: MUSVIの窓でつながれる専門家・実践者・企業人のリスト。生徒はカタログを見て「この人に会いたい」とリクエストできる。窓の教育コーディネーターが接続を調整し、等身大の遠隔対話で多様な大人との出会いを実現する。地理的制約を超えた「出会い」のインフラ。

ZINE・イコールが出会いの入口になる: 先輩たちや他校の人たちが作ったZINEは、次の世代にとって「探究のカタログ」。雑誌『イコール』やイコールZINEは、社会福祉・法律・歴史・心理学・デザイン・メディアなど多領域の執筆者が書いている。生徒は記事を読んで興味を持った執筆者に、取材やゲスト登壇を依頼できる。教師が用意したゲストではなく、生徒自身の関心に基づいてゲストとつながる。

問いかけの支援: 「なぜそれが気になるの？」「それを知ったらどうしたいの？」と問いかけ、違和感の奥にある本人の関心の核を一緒に探る。

❷ 掘る——テーマの解像度を上げる

コンピテンシー

4. 知の形式知化と接続 / 2. 越境学習 / 6. 合意形成と共創

AIによる背景の掘り下げ: 生徒は一次情報（自分の足で集めたデータ）を持っている。AIはそれに二次情報（文献・統計・他地域の事例）を接続する。

MUSVIの窓: 単なるビデオ会議ではなく「解像度の高いコミュニケーション」。離島や過疎地域の生徒にとって、多様な他者との出会いの質を保証するインフラ。

❸ 調べる——先行事例研究と仮説構築

コンピテンシー

3. パターン発見 / 5. 仮説駆動 / 2. 越境学習

核心的な姿勢

「すでに誰かがやっていないか？」をまず調べる。あれば学ぶ。なければ自分で実験して検証する。

先行事例の対象: 同じ課題に取り組んだ地域の事例 / 類似テーマの他校の探究成果 / 企業・NPOの実践 / 論文・書籍・報告書 / 海外の事例（AIが言語の壁を解体）

育成する姿勢: 「車輪の再発明をしない」——先人の知恵を探し、その上に立つ。「なければ自分で作る」——先行事例がないこと自体が探究の価値。

調査の伴走: 先行事例の探し方、AIを使った調査の質の高め方を伴走する。

❻ 響く——社会的反響と次のサイクル

コンピテンシー

7. 意味の自己生成 / 6. 合意形成と共創 / 3. パターン発見

自分が制作したものが他者に読まれて影響が生じる体験:

  • 「あの記事読んだよ」と地域の人に声をかけられる
  • 他校の生徒が自分のZINEを引用して新しい探究を始める
  • 取材先の当事者から「変化が起きた」と報告が来る
  • 読者から反論や別の視点が届く

ZINEを通して相互に影響を与え合いながら学んでいく環境が生まれる:

  • 後輩がZINEを読んで新しいテーマと出会う（→ ❶へ）
  • 他校のZINEと自分のZINEが干渉して新しい問いが生まれる
  • 地域の課題がZINEを通じて可視化され、行政や企業が動き始める

キャリア探究への接続: 社会的反響を受けた後、Vaultに蓄積された探究の軌跡とフィードバックを手がかりに、「この経験を通じて、自分は何をしたい人間だと分かったか」を掘り下げる。これがキャリア探究であり、総合型選抜の志望理由書の核になる。
